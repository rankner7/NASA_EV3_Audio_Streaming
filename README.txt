- ITU G.729 Source Code
	-Installation: 
	 - Download zip file from: https://www.itu.int/rec/T-REC-G.729
	 - Extract
	 - Choose G.729 version you would like to use and go to "c_code" folder
	 - Make file names lower case if they are not -> use make_lower_case.py tool in "Personal Tools"
	 - edit coder.mak and decoder.mak to reflect your system's compiler (most likely gcc)
	 - run "make -f coder.mak" and "make -f decoder.mak"
